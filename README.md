# Megadraft

Rich Text editor built on top of draft.js (Work in Progress)

## Developing

To run the development server and see the examples:

```
git clone <REPO_URL>
make setup
make run
```

Then visit localhost:8080 on your browser.

To run local tests:

```
make test
```

To lint local source files:

```
make lint
```
